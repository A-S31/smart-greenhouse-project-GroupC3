# Smart Greenhouse Project

An Arduino-based automated greenhouse monitoring and control system built as part of our sensors course. The system uses multiple sensors to track environmental conditions — temperature and humidity (DHT11), soil moisture, ambient light (LDR), plant/container weight (HX711 load cell), and water flow for irrigation (YF-S201) — and uses a relay module to automatically control a water pump and fan based on real-time readings. Live status is displayed on an onboard LCD screen.
This repository contains the full technical report covering system design, sensor selection, calibration procedures, experimental results, and engineering constraints. Also a demonstration video is present.
